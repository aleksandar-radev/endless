# Endless

Endless is an idle RPG where you fight enemies, collect loot, upgrade your hero, and keep progressing even in long sessions.

## Download

Download the latest version from the project's **Releases** page on GitHub.

## Run on Your Platform

### Windows
- Download the `.exe` installer.
- Run the installer, then launch **Endless** from Start Menu or desktop shortcut.

### macOS
- Download the `.dmg` package.
- Open it and drag **Endless** into Applications.
- On first launch, macOS may show a security warning for unsigned apps. If needed, open **System Settings > Privacy & Security** and allow the app.

### Linux
- Download the Linux package from Releases (for example `.snap` if provided).
- Install it with your distribution's package/app installer.
- Launch **Endless** from your app menu.

## Save Data

- Your progress is saved locally on your device.
- Reinstalling or changing devices can remove local progress unless you backup saves first by logging in.

## Troubleshooting

- If the app does not start, re-download the latest release and install again.
- If your OS blocks launch, use the system's security prompt to allow the app.
- If you encounter a bug, open an issue on GitHub with your OS, game version, and steps to reproduce.
